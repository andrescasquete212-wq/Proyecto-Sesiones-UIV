# Proyecto UIV - Variables de Sesión en GitHub

Este repositorio corresponde a la actividad **UIV: Elaborar un proyecto con variables de sesiones en la plataforma GitHub**.

## 📂 Contenido
- Documentación del proyecto
- Ejemplo de manejo de ramas en Git
- Archivos de prueba para simular variables de sesión

## 🚀 Pasos realizados
1. Inicialización del repositorio con `git init`.
2. Creación de la rama principal `simon`.
3. Configuración del remoto en GitHub.
4. Primer commit con archivos de ejemplo.
5. Push al repositorio remoto.

## 📌 Objetivo
Demostrar el uso de **Git** y **GitHub** para gestionar un proyecto con sesiones, incluyendo:
- Creación de ramas.
- Manejo de commits.
- Conexión con repositorio remoto.
- Documentación en README.

## 📝 Notas
Este proyecto se centra en la práctica de **versionamiento** y **gestión de repositorios** en GitHub, más que en el desarrollo de código específico.
